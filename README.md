## Installation & setup
1. Click the link to open up my GitHub page. 

https://github.com/clounsbu/Python-to-do-list

2. Once on the GitHub page click on the **green** button to the right that says **code**. This will bring a dopdown menu, you will click on **HTTPS** and copy the link below it. 
3. Open you terminal in VS code and type
```sh
git clone https://github.com/clounsbu/object-oriented-python.git
```
4. You will see a folder named **object-oriented-python**. Open the folder and open the folder in **Visual Studio Code**.
5. To view the project open the **practice.py** file in VS Code.
6. Once the file is open on the top right hand corner you will see what loops like a play button with an arrow to the right if it. Click the arrow and you will be presented with a drop down menu, click on run Python file to view the project in the terminal.

## How to use as a typical user
- After you click on the play button you will see the code output in the terminal. 
- You will first be presented with the students information including their name, email, grades, and average grade.
- Below the student info you will see a section called search by email where you can enter a students email. Enter a student email and if that student exists you will see a message saying student found! followed by their information. If the student email doesn't exist you will see a message saying student not found. 
- After you hit enter after submitting the student email you will be presented with a section for tuples and list operations.
